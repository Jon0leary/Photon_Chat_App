# Photon Chat App

Photon Chat App is a Unity networking prototype demonstrating real-time multiplayer chat using the Photon networking framework.

The project explores how multiplayer communication systems can be implemented inside a Unity environment, focusing on connecting players to shared rooms and exchanging messages in real time.

## Tech Stack

- Unity
- C#
- Photon Networking

## Features

- Real-time chat communication between players
- Connection to Photon multiplayer rooms
- Network message synchronization
- Foundation for multiplayer gameplay systems

## Purpose

This project was created to explore multiplayer networking concepts in Unity and to serve as a foundation for future multiplayer game mechanics.
